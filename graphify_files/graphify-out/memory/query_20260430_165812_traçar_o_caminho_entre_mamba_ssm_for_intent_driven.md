# Q: Traçar o caminho entre Mamba-SSM for Intent-Driven Networks e LLM for Intent Extraction ou NWDAF

## Answer

Caminho até LLM for Intent Extraction: Mamba-SSM for Intent-Driven Networks -> Hierarchical Decision Transformer with Goal Awareness (HDTGA) -> QLoRA-tuned LLM for Intent Processing -> LLM for Intent Extraction. O primeiro salto é uma similaridade semântica inferida com HDTGA; o segundo é uma relação conceitual explícita; o terceiro é outra similaridade semântica inferida. No grafo atual não existe caminho entre Mamba-SSM for Intent-Driven Networks e Network Data Analytics Function (NWDAF).
